# Sistema de Gestión de Mascotas

## Descripción

Este proyecto implementa un sistema básico de gestión de mascotas utilizando dos enfoques de programación:

1. Programación Tradicional.
2. Programación Orientada a Objetos (POO).

El sistema permite registrar y mostrar la información básica de una mascota, incluyendo:

* Nombre
* Especie
* Edad
